# Security

Report security issues through GitHub private vulnerability reporting if available, or by opening a minimal issue that does not disclose exploit details.

This plugin contains local Node.js scripts. Users should inspect scripts before running them, especially when using a fork.

The included scripts are intended to be read-only, dependency-free, and local-only. They should not exfiltrate data, require network access, collect telemetry, or modify project files.

There is no telemetry by default.
